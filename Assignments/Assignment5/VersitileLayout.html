About This Map

This interactive map highlights a variety of points of interest across Washington D.C., including theaters, bars, bicycle rentals, and music venues. Users can explore the city by panning and zooming, and toggle categories on and off using the checkbox legend in the top-right corner of the map.

Each icon on the map represents a specific type of location. Clicking on an icon opens a popup with information about that place’s category. Use the checkbox legend in the top-right corner of the map to show or hide different point categories.

The point data is loaded in GeoJSON format and styled using Mapbox GL JS, a JavaScript library for interactive web maps. This versatile layout was originally created by Andy Cochran and allows for reactive resizing of page elements depending on screen size.
